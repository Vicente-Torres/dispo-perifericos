# Bluetooth Arduino
## Informações
O aplicativo conecta-se à um dispositivo bluetooth e permite o envio de mensagens do aplicativo para o dispositivo.

Dentro da pasta 'dist/' desse diretório está localizado o arquivo .apk já buildado. <br/>
Há também o programa .ino de exemplo, utilizado por um Arduino Uno R3 para receber as mensagens do aplicativo.
